# Docs

Este directorio contiene documentación adicional sobre el proyecto.

## Estructura del directorio:

- `database-design/`
- `propuesta/`

## Propuesta

Contiene la propuesta de diseño del sistema, además de los requisitos iniciales y la visión general
del proyecto.

### Archivos

- `propuesta.md`: Archivo principal de la propuesta de diseño.
- `propuesta.pdf`: Documento PDF generado a partir de `propuesta.md`.
- `Requisitos del sistema.pdf`: Documento PDF original de los requisitos del sistema.
- `VisionUPEDU.pdf`: Documento PDF de la plantilla de visión del proyecto.

> Nota: Para generar la documentación de la propuesta, se puede editar el archivo `propuesta.md` y
> ejecutar
> `pandoc propuesta.md -o propuesta.pdf --from markdown --template eisvogel --pdf-engine=lualatex`.
